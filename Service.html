Static Website

tatic web pages are made of “fixed code,” and unless the site developer makes changes, nothing will change on the page. Think of it like a brochure for a business. Static sites give a lot of the same type of information that you could get from a brochure

Responsive Web Design

Responsive Web Design is about using HTML and CSS to automatically resize a website.Responsive Web Design is about making a website look good on all devices (desktops, tablets, and phones)

Customisable Layouts

Custom templates are best used for pages that don't follow the normal template. Their purpose is to allow you to create a custom layout to use only on specific pages.
